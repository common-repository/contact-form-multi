== Installation ==

1. Upload the `contact-form-multi` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Create a page or a post and insert the shortcode [contact_form id=you_form_id] into the text.

== Frequently Asked Questions ==

= How can I add a new contact form? =

1. Please go to the Contact Form plugin main page and click the + button.
2. Fill out all the fields that are necessary to be displayed in the form on a page or in a post.
3. Click the Save button.

= How can I change the title of the form's tab? =

1. Click the tab the title of which you want to be changed.
2. Enter the new title and click anywhere on the form.

= How can I delete a form? =

1. Click the Trash icon in the tab of the contact form that you want to delete.
2. After clicking the Trash icon you will see a pop-up window with a question if you still want to delete the contact form. If you haven't changed your mind to delete the form, please click OK, otherwise click Cancel.

= How can I add a contact form to my site? =

Please insert the shortcode [contact_form id=*] to a post or page.

= I have some problems with the plugin's work. What Information should I provide to receive proper support? =

Please make sure that the problem hasn't been discussed yet on our forum (<https://support.bestwebsoft.com>). If no, please provide the following data along with your problem's description:

1. the link to the page where the problem occurs
2. the name of the plugin and its version. If you are using a pro version - your order number.
3. the version of your WordPress installation
4. copy and paste into the message your system status report. Please read more here: [Instruction on System Status](https://bestwebsoft.com/documentation/admin-panel-issues/system-status/)
